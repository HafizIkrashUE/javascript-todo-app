# To-Do App

A simple and responsive To-Do List application built using HTML5, CSS3, and Vanilla JavaScript.

## Screenshot

![To-Do App]
<img width="955" height="491" alt="Screenshot" src="https://github.com/user-attachments/assets/3b8aaec1-bfd1-414b-8889-62677bd833c6" />



## Features

- Add new tasks
- Edit tasks by double-clicking
- Mark tasks as completed
- Delete tasks
- Save tasks using Local Storage
- Responsive design

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## JavaScript Concepts Used

- Arrays
- Objects
- Functions
- DOM Manipulation
- Event Listeners
- createElement()
- appendChild()
- Local Storage
- JSON.parse()
- JSON.stringify()
